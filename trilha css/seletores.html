Seletores CSS

  • Seletor por tipo: Busca elementos por uma Tag HTML especifica. Ex: "h1{}", "div{}",
  • Seletor por ID (#): Busca elementos atraves do atributo "id"
  • Seletor por Classe (.): Busca elementos atraves do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor por Atributo([atrib]): Seleciona os elementos que possuem um atributo especifico em nosso documento HTML e conseguimos tambem buscar atributos com um valor especifico
